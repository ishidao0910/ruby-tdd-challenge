# ruby-tdd-challenge
this repo is for mixi's TDD challenge at 7/16/2022

# BUILD
```
docker build .
```

# TAGGING
```
docker tag ee2b00afc52f ruby/tdd:3.1.2
```

# RUN CONTAINER 
```
docker run -it -d --name ruby_tdd ruby/tdd:3.1.2
```

# ENTER THE CONTAINER
```
docker exec -it ruby_tdd ash
```
